# Download Issues

> The download won't start or gets blocked

If clicking the download button does not start the download, try the following solutions.

## Try the manual download link

When the download dialog appears, click the **manual download link** to download directly.

## Check your browser

Your browser may be blocking the download. Look for:

- A blocked download notification in the browser's **download bar**
- A popup or notification in the **top-right corner** of the browser
- A shield icon in the **address bar** (especially in Chrome and Edge)

If you see a blocked download notification, click **"Keep"** or **"Allow"** to proceed.

## Try a different browser

If the issue persists, try downloading with a different browser:

- [Google Chrome](https://www.google.com/chrome/)
- [Mozilla Firefox](https://www.mozilla.org/firefox/)
- [Microsoft Edge](https://www.microsoft.com/edge)

## Download from GitHub directly

You can always download optimizerDuck directly from the [GitHub Releases page](https://github.com/itsfatduck/optimizerDuck/releases).

## Check your internet connection

Make sure you have a stable internet connection. Try:

1. Opening other websites to verify connectivity
2. Restarting your router
3. Disabling VPN if you are using one
