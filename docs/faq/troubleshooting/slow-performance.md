# App Running Slowly

> optimizerDuck is unresponsive or slow

If optimizerDuck takes a long time to load or respond, try the following.

## Close unnecessary programs

Other applications may be consuming system resources. Close any programs you are not actively using, especially:

- Web browsers with many tabs open
- Games or resource-intensive applications
- Other system tools running in the background

## Check your disk space

If your system drive (C:) is nearly full, your entire system (including optimizerDuck) may become sluggish:

1. Open **File Explorer**
2. Right-click your **C: drive** → **Properties**
3. If free space is below **1 GB**, free up some space

## Restart your computer

A simple restart can resolve many performance issues by clearing temporary files and freeing up memory.

## Disable unnecessary startup programs

Too many startup programs can slow down your system. You can use optimizerDuck's built-in [Startup Manager](/docs/guides/startup-manager) to manage which programs run at startup.
