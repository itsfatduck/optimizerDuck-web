# Community

Join the optimizerDuck community to get help, share feedback, and connect with other users.

## Discord

Our **Discord server** is the best place to interact with the community and the development team.

<DiscordWidget guildId="1091675679994675240" />

### What you can do on Discord

- **Get help** with any issues you encounter while using optimizerDuck
- **Share your feedback** and suggestions for improving the app
- **Report bugs** and get real-time support from the development team
- **Stay updated** on the latest news, releases, and upcoming features
- **Chat with other users** who are also optimizing their Windows systems

### How to join

1. Click the **Discord invite link**: [discord.gg/tDUBDCYw9Q](https://discord.gg/tDUBDCYw9Q)
2. Create a Discord account if you don't have one already
3. Accept the server rules
4. Start chatting!

## GitHub

You can also engage with the project on [GitHub](https://github.com/itsfatduck/optimizerDuck):

- ⭐ **Star the repository** to show your support
- 🐛 **Report bugs** by [creating an issue](https://github.com/itsfatduck/optimizerDuck/issues/new/choose)
- 💡 **Request features** by [opening a feature request](https://github.com/itsfatduck/optimizerDuck/issues/new/choose)
- 🔧 **Contribute code** by submitting a pull request
